# Rise at Seven — Clone

A clone of [riseatseven.com](https://www.riseatseven.com) built with React, Tailwind CSS & Framer Motion.

**Live Demo:** [rise-at-seven-clone-neon.vercel.app](https://rise-at-seven-clone-neon.vercel.app/)

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion

## Run Locally

```bash
npm install
npm run dev
```
